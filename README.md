# pkg-lint

package.json quality checker. 25+ checks across 7 categories with A-F grading and fix suggestions.

**[Live Demo →](https://yurukusa.github.io/pkg-lint/)**

## Features
- 25+ checks for completeness, best practices, and common mistakes
- Grade A-F scoring across 7 categories
- Actionable fix suggestions for every issue
- Paste or drag-and-drop your package.json

## Usage
Paste your package.json content and get instant feedback with a quality grade and specific recommendations.

## Tech
- Single HTML file, vanilla JavaScript
- No dependencies, no build step
- Works offline after first load
